# ``Prometheus``

A prometheus client library for Swift.

## Overview

``Prometheus`` supports creating ``Counter``s, ``Gauge``s and ``Histogram``s and exporting their
values in the Prometheus export format. 

## Topics

### Collectors

- ``Counter``
- ``Gauge``
- ``Histogram``
    - ``Bucketable``
- ``DurationHistogram``
- ``ValueHistogram``
